Pomodoro App 🍅

Aplicativo de timer Pomodoro para Windows desenvolvido em Python com a biblioteca Tkinter. 💻

Funcionalidades: ✨
Ciclos de 25 minutos de foco. 🧠
Pausas de 5 minutos. ☕
Alternância automática entre os modos de foco e pausa. 🔄
Interface com tema escuro. 🌙

Como executar o código-fonte: 🚀
1. Instale o Python no seu sistema. 🐍
2. Clone este repositório no seu computador. 📂
3. Abra o terminal na pasta do projeto e execute o comando: python pomodoro.py. ⌨️

Como utilizar o executável: 📦
Acesse a seção Releases na página do repositório. 🔗
Faça o download do arquivo pomodoro.exe. ⬇️
Abra o arquivo. A instalação não é necessária. ✅
